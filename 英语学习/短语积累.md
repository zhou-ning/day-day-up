翻译题、作文当中比较好的短语积累

### 关于中文

* 用作：

  ```
  ... is used as ...
  ```

* 本族语、本土语

  ```
  native language
  ```

* 显著区别

  ```
  significant distinction
  ```

* 同一形式

  ```
  in a unified form
  ```

* 对。。。发挥了重要作用

  ```
  play a crucial part in...
  play an important role in
  play a vital role in ...
  ```

* 中华民族团结

  ```
  unification of the Chinese nation
  ```

* 经济快速增长

  ```
  the rapid growth of economy
  ```

* 全球化影响增加

  ```
  the enhancement of global influence
  ```
